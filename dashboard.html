Problem Statement
Modern vehicle rental and fleet management systems face two major inefficiencies:
  • Sub-optimal vehicle relocation — moving cars from drop-off to high-demand pickup points without optimal routing.
  • Static pricing models — rates that fail to adapt to real-time demand, weather, seasonality, or competitor pricing.
This system solves both by combining A* pathfinding for optimal routing with machine learning regression for dynamic pricing.
Task 1 — A* Algorithm
Selected: A* (A-Star) Search
Why: Guarantees optimal path with admissible heuristic
Heuristic: h(n) = Euclidean distance to goal
Graph: 18 Karachi nodes, 31 bidirectional edges
Complexity: O(E log V) with min-heap open set
Performance: Finds optimal path in <1ms
Task 2 — ML Regression
Type: Regression (continuous price output)
Dataset: 6,500 synthetic records (12 features)
Split: 80% train / 20% test + 5-fold CV
Models: Random Forest & Gradient Boosting
Winner: Gradient Boosting (RMSE $4.04, R² 0.9862)
Preprocessing: OneHotEncoding + StandardScaler
